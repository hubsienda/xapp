# QBX.app

Lean multilingual browser utility for checking business list quality, category chaos, vendor complaint preparation, and campaign readiness.

Doctrine for version 1:

1. No login, accounts, dashboard, database, subscriptions, or backend file processing.
2. CSV/XLSX files are analysed locally in the browser.
3. English UK, Spanish Spain, and Italian are supported from version 1.
4. Paid products are external toolkits with URLs controlled in `src/config/site.ts`.

## Development

```bash
npm install
npm run dev
npm run build
```
